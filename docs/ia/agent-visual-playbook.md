# JRAG Agent Visual Playbook

Use this file when building pages, components, or layouts.

## Visual Mood

The site should feel:

- editorial
- cinematic
- industrial
- luxurious
- restrained

The site should not feel:

- generic corporate
- beachy
- overly romantic
- playful
- app-like

## Typography Roles

- `Quicksand` for hero headlines, H1s, major section titles, standout product names
- `Lato` for subheads, intros, support lines, transition text
- `Roboto Condensed` for paragraphs, nav, labels, buttons, forms, data, specs, metadata

## Surface Rules

Use strong contrast between sections.

- dark/image-led openings for drama
- white and sand sections for breathing room
- orange only for action or key emphasis
- blue for trust, information, and system cues

Do not make the site one flat tone.

## Composition Rules

- prefer large, confident sections over crowded modules
- keep text columns disciplined and readable
- use asymmetry where it helps hierarchy
- let big visuals carry emotional weight
- let proof blocks feel narrower, tighter, and more structured than story blocks

## Component Types

Use a small recurring set of visual modes:

- `story` = atmosphere, image, narrative
- `proof` = certifications, traceability, logistics, process
- `route` = audience choice or next-step navigation
- `metric` = one strong fact or number
- `editorial` = article, recipe, or discovery feature

## Header and Navigation

- navigation should feel slim and premium
- audience switching must stay obvious
- nav text should feel precise, not decorative
- the header should not look like a SaaS dashboard

## Motion

- motion should be smooth, quiet, and deliberate
- fade-up and line-draw behaviors are good
- avoid bouncy, gimmicky, or toy-like animation
- hover states should sharpen or reveal, not jump around

## Page Balance Rule

Important pages need both:

- one noble moment
- one rigorous moment

Examples:

- a product page can pair a beautiful shellfish image with a traceability or caliber block
- a B2B page can pair an operational proof section with a strong lagoon/product visual

Utility and legal pages can express the noble side more lightly through spacing, typography, and polish.

## Photography Direction

Three image modes:

- product as jewel
- process as lab
- environment as monument

Image use should be confident and curated.
Prefer fewer, larger images over many small decorative ones.

## Accessibility Reminder

`Roboto Condensed` is approved as the reading layer, so readability must be protected with:

- good contrast
- enough line-height
- disciplined paragraph width
- clear hierarchy on mobile

## Anti-Patterns

Do not make JRAG look like:

- a luxury hotel website
- a generic Shopify seafood theme
- a startup landing page
- a cold B2B portal with no atmosphere
- a lifestyle brand with no proof
